## HFZF

Hierarchical Fuzzy Finder.

## Installation

```
$ go get -u github.com/kitagry/hfzf
```

## Usage

You can use this command to input file name

```
$ hfzf YOUR_YAML_FILE
```

You can also use to input yaml data from standard input.

```
$ cat YOUR_YAML_FILE | hfzf
```
